Intro
This is a simple, working Specflow-Watin example, which stores all groupon deals of a given user in a file.

Build Status: 

Windows Requirements
- Internet Explorer 8.0, Visual Studio 2008 and above, Specflow and Watin 2.0

Run
- Goto GetDeals.cs file and edit the username and password values for your groupon account and update the relevant full names in the test.
- Run the code

Results
- A .txt file with list of all your deals will be saved in the TestResults\<Testmachine Test Time> directory.
